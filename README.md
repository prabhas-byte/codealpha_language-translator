# Language Translator

Real-time multilingual text translator with speak & copy features – built with pure HTML/CSS/JS.

![image alt](https://github.com/prabhas-byte/codealpha_language-translator/blob/ce58ae19ee186d6e0ed7a007ec74ea806baf8e8a/Screenshot%202026-02-02%20151202.png)

## Features

- Auto-detect source language
- Supports 20+ languages (English, Telugu, Hindi, Tamil, Arabic, French, Spanish, etc.)
- Instant translation using MyMemory API (free tier)
- Copy translated text to clipboard
- Text-to-speech for translated output (browser speech synthesis)
- Responsive glassmorphism + neon UI
- Loading states & error handling

## How to Run

1. Open `index.html` in any modern browser (Chrome/Edge recommended)
2. Type or paste text
3. Select source/target languages (or leave auto-detect)
4. Click "Translate"
5. Use "Copy" or "Speak" buttons

## Technologies

- HTML5
- CSS3 (glassmorphism, gradients, flexbox, animations)
- Vanilla JavaScript (ES6+)
- MyMemory Translation API
- Web Speech Synthesis API
